# 小笛手 · 使用手册

> 适合 6-12 岁小朋友使用的竹笛练习 App

---

## 一、认识界面

打开 App 后，你会看到**主练习页**，顶部有两个按钮：

| 按钮 | 功能 |
|------|------|
| 🏋️ 专项 | 进入专项节奏练习 |
| 📹 录像 | 录制练习视频打卡 |

---

## 二、自由练习（主页）

### 怎么用？

1. 拿起竹笛，对准手机麦克风
2. 点击屏幕中间的大 **▶ 按钮** 开始
3. 对着曲谱吹奏《小星星》
4. 上方圆圈会跟着你吹的音符高亮移动
5. 中间的**音准仪表盘**会实时显示你吹的音是否准确
6. 吹完后点 **⏹ 按钮** 停止
7. 自动跳转到**结果页**，看看你的得分！

### 结果页说明

| 图标 | 意思 |
|------|------|
| ⭐ 完美 | 音准误差在 ±25 音分以内，非常棒！ |
| 👍 良好 | 音准误差在 ±50 音分以内，不错！ |
| 🎯 偏差 | 音符吹对了但音准偏了，再调整一下 |
| 😅 漏音 | 这个音没有吹出来 |

---

## 三、专项练习

点击顶部 **🏋️ 专项** 进入课程列表，共有 5 个练习：

| 练习名 | 节奏型 | 练习目标 |
|--------|--------|----------|
| 8个短音 | ♩♩♩♩♩♩♩♩ | 气息均匀、音头清晰 |
| 4短一长 | ♩♩♩♩♩— | 气息控制与音符时值 |
| 长短交替 | ♩— ♩ ♩— ♩ | 音符时值对比 |
| 附点节奏 | ♩. ♪ ♩. ♪ | 附点节奏感 |
| 切分节奏 | ♪ ♩ ♪ | 重音位移感 |

### 练习流程

```
选择课程 → 3秒倒计时 → 开始吹奏 → 自动结束 → 查看得分
```

### 练习界面说明

- **大字音符**：显示你当前吹的音，绿色 ✅ 表示音准了
- **节拍方块**：橙色高亮 = 当前拍，✓ = 已完成的拍
- **进度条**：绿色进度条显示练习进度
- 吹完所有拍子后自动跳转结果页

### 得分说明

| 分数 | 评价 |
|------|------|
| 90+ | 🏆 太棒了！ |
| 75-89 | 🌟 很不错！ |
| 60-74 | 👍 继续加油！ |
| 60以下 | 💪 再练练！ |

综合分 = 音准分（60%）+ 节奏分（40%）

---

## 四、录像打卡

点击顶部 **📹 录像** 进入录像页：

1. 点击录制按钮开始录像
2. 对着镜头吹奏
3. 点击停止，App 会自动合成带日期水印的视频
4. 保存到相册，分享给爸爸妈妈看！

---

## 五、小贴士

- 🎵 练习时保持安静的环境，效果更好
- 📱 手机麦克风朝向笛子吹口
- 🌬️ 吹奏前先做几次深呼吸
- 🔁 每个练习多练几遍，进步会很快！
